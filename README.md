# Media_Markt
Dit project is een poging om de gebruikersinterface en functionaliteit van de MediaMarkt-website na te maken. De website is gebouwd met Laravel voor de back-end en Tailwind CSS voor de front-end. Het doel is om delen van de MediaMarkt-site na te maken, zoals het bekijken van producten, het winkelwagentje, inloggen en het bestelproces.
